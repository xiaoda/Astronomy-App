# MVP 功能清单（无声音版）

## 目标

- 用户打开网页后 3 秒内进入“宁静星空”体验。
- 全程无学习成本，无登录、无复杂操作。

## P0 必做（第一版就做）

1. **单页直达体验**

- 只有一个页面，打开即进入全屏星空。
- 无启动页、无注册登录、无弹窗打断。

2. **一个主场景（90 秒循环）**

- 深色宇宙背景 + 星点缓慢闪烁 + 轻微漂移。
- 动画速度统一偏慢，避免快速/突兀变化。

3. **极简交互（只保留 2 个）**

- `轻触`：切换一句短文案（宁静引导语）。
- `长按 1 秒`：触发一次流星动画。
- 不做菜单层级，不做复杂设置。

4. **文案系统（静态）**

- 预置 10-20 条短句，随机展示。
- 每次只显示 1 句，3-5 秒淡入淡出。

5. **视觉呼吸模式（可选开关，默认开）**

- 星空整体亮度按慢节奏微弱起伏（如 4 秒吸气/6 秒呼气节奏）。
- 仅视觉，不包含声音。

6. **基础设置入口（1 个按钮）**

- 右上角“简洁设置”：仅包含
  - 文案显示：开/关
  - 呼吸模式：开/关
- 不做账号同步，不做个性化配置。

7. **移动端优先适配**

- 支持主流手机竖屏（iOS Safari / Android Chrome）。
- 首屏可交互时间 < 3 秒（4G/Wi-Fi 常规环境）。

8. **基础性能与稳定性**

- 动画目标帧率：中端手机尽量接近 60fps，最低不低于 30fps。
- 页面切后台再回来后动画可恢复，不黑屏不卡死。

## P1 可后续再加（先不做）

- 多场景切换（星云/月面/银河）
- 收藏喜欢文案
- 每日主题
- 声音与环境音设计

## 建议技术实现（简单可落地）

- 前端：`Vite + React`（或纯 `Vanilla JS`）
- 动画：先用 `Canvas 2D`（MVP 足够，开发快）
- 样式：CSS 变量管理主题色与动画节奏
- 部署：`Vercel` 或 `Netlify`
