# Project Rules

## UI Language
- 整个 APP 的默认界面文案、提示语、按钮文字、表单标签、占位符、空态文案、无障碍标签一律使用中文。
- 只有明确的产品标题、曲目名、品牌名、库名、API 名、文件名等专有名词可以保留原文。
- 如果新增文案拿不准是否应保留英文，默认先写中文，再在必要处补充原文。

## Copy Editing
- 修改功能时，不要把现有中文界面文案替换成英文。
- 新增设置项或交互提示时，优先复用现有中文表达风格，保持简洁、自然、统一。

## Scope
- 这些规则适用于 `src/` 下所有用户可见文本。
- 文档、代码标识符、类型名、组件名、脚本命令不受此限制，但如果它们直接展示给用户，仍应提供中文界面文案。
